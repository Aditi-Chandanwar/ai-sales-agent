# AI Sales Agent

## Problem

Industrial sensor distributors spend significant time manually identifying leads, reaching out to prospects, and understanding customer requirements.

## Goal

Build an AI-assisted workflow that helps:

- Discover leads
- Extract public contact information
- Automate outreach
- Analyze customer responses
- Recommend relevant products

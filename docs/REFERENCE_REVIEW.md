# Reference Project Review

## Useful ideas to borrow

1. Email outreach structure

- Separate email drafting from email sending
- Human approval before sending

2. Reply handling

- Future module for reading replies
- Summarize customer requirement before response

3. Reporting

- Generate summary report after pipeline run

4. Hybrid qualification

- Rule-based filter first
- LLM review only for Review leads

## What we will not copy

- We will not replace our existing architecture
- We will not merge folders directly
- We will not use one flat script-based structure
- Our repository remains the source of truth
